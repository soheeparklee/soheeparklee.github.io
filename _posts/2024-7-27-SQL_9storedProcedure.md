## ✅ Stored Procedure

> run several queries as if it is one function

### 👍🏻

- optimization, cache
- easy to update
- less traffic between client and server with SQL
- security

### 👎🏻

- less compatibility
- slower than C, JAVA in caculation
- difficult to debug
