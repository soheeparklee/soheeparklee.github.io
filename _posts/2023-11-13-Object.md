# Function

## Function & Object

> console.log으로 보여주는 함수 만들기

```javascript
    const caculator= {
        add: function (a, b) {
            console.log("a+b" = a+b);
        },
        //화살표 함수
        minus: (a, b) => {
            console.log("a-b" = a-b);
        }
    }

caculator.add(10+20)

```

30

> return 은 값을 반환\
> 그래서 그 값을 변수에 저장하였음

```javascript
const caculator = {
  add: function (a, b) {
    return a + b;
  },
  //화살표 함수
  minus: (a, b) => {
    return a - b;
  }
};

const result = caculator.add(10 + 20);
console.log(result);
```
