## ✅ to use scanner import

```java
    Scanner sc = new Scanner(System.in);
```

## ✅ string

```java
String input = sc.nextLine();
```

## ✅ other datatype

```java
boolean bool = sc.nextBoolean();
int intNum = sc.nextInt();
double dblNum = sc.nextDouble();
```
