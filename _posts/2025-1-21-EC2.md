## ✅ EC2에 mySQL install

- update위해 쓰이는 명령어

```
sudo yum update -y # For CentOS/RHEL based systems

sudo vi /etc/my.cnf #configure 파일 수정
                    #   bind-address = 0.0.0.0 수정 위해 필요

sudo systemctl restart mysqld
sudo systemctl status mysqld
```
