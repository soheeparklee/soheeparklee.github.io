## ✅ Operador

```java
int i=1;
int z;

z = 5 + i++; //z=6
z = 5 + i;
i++;

z= 5 + ++i; //7
i++;
z = 5 + i;
```

## ✅ Conversiones, Casting

1. Conversiones implicitas

```java
int x = 5;
double y;

y = x;
//possible, double > int
```

```java
int x ;
double y = 5.3;

x = y;
//not possible, double > int
```

2. Conversiones explicitas = casting

```java
int x ;
double y = 5.3;

x = (int) y;
//now possible
// x = 5 , trunca
```
