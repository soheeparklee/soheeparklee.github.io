```java
    public static void main(String[] args) {
    }
```

`String[] args`
🟰 JAVA Program Arguments
🟰 Command Line Arguments
🟰 자바 명령 매개변수

자바 실행 시, 외부에서 전달받는 변수
Command Line(terminal)에서 주는 명령이 args에 저장되어서 이름이 명령 매개변수임.
